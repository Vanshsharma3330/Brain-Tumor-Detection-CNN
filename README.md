# Brain Tumor Detection using CNN

This project implements a Convolutional Neural Network (CNN) to classify brain MRI images into four categories:
- Glioma
- Meningioma
- Pituitary Tumor
- No Tumor

The model is trained using deep learning techniques and evaluated on unseen MRI images to assist in automated brain tumor detection.
